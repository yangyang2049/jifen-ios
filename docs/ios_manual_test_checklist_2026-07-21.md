## 8. 手机 ↔ 手表联动（发布阻断回归）

真机：配对 iPhone + Apple Watch，双方安装同一构建。每项勾选一次即可。

| 检查项 | 结果 | 备注 |
| --- | :---: | --- |
| 前台联动：手机发送 → 手表确认 → 手表计分 → 自动完赛 | ⬜ | 手机仅 1 条 finished 记录；时长/比分真实；标「手表记录已同步」 |
| 菜单手动结束（含射箭） | ⬜ | 手机收到 `matchFinished` 并退出跟随；无草稿脏落库 |
| 台球（黑八/追分/斯诺克）手动结束 | ⬜ | 同上 |
| 锁屏 / 手表暂时不可达后恢复 | ⬜ | snapshot ACK 重试；完赛记录仍能回传 |
| 手机接管后再完赛 | ⬜ | 不产生重复手表记录；follower 退出不额外 persist |
| 投篮训练完赛 | ⏸ | 仅手表本地；不回传手机（预期） |
